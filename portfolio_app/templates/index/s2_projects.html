Yomama

A lot of times communicating using technology with moms/elders who are not exactly tech savvy is SUPER HARD! We wanted to create sometime so simple, helpful and beautiful for our moms and elders to use; from simple communication all the way to serious emergencies.

The app has 3 main features:

Emergency - which is a red button that pings the location to ask for help!

I'm Lost - which is a yellow button that pings the location to tell people where they are at.

We also added two ten character words that they could add for messaging.(It defaults to "dinner" and "I love you")

The app also has a dashboard panel to view past communications with map locations and ability to change the two 10 character words.

Technologies: Python, Django, jQuery, PostgreSQL, YO API, AJAX

Course Facilitation Tool

This tool is meant to improve real-time teacher-student communication in class with my collaborates.


Student operations:

Done - no problem, Help - ask for teacher's immediate attention, and Question - raise questions for the current slide.


Teacher operations:

Count and handle the class progress by the statistics of students' operations. Respond to individual issues.


The project helped students raise questions or ask for help by a single click, and helped teachers control the teaching efficacy.

Technologies: Python, Django, jQuery, reveal.js, AJAX, AWS

Let's Party

How do you find something for fun? Life is hard, party is easy.

This AngularJS-based project aimed to let user search and share party information.

Functions include:

You: save your favorite parties, show the parties by your host or favorites.

Browse: search the parties close to you, search with filters, show the party info that qualified.

My final project in RocketU.

Technologies: AngularJS, Django Rest Framework, Python, Django, PostgreSQL, Google Map API, AJAX, AWS
